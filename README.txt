
JDE Luxury Rental — Site Core (no images)

This package contains the whole site structure without heavy car photos.

1) Upload the CONTENTS of this zip to your GitHub repo root (not the zip itself).
2) In GitHub Pages settings: Deploy from a branch → main → /(root).
3) Add your photos later: put them into /assets/cars/ and use these exact filenames:
   - ferrari-812-superfast.jpg
   - ferrari-f8-spider.jpg
   - ferrari-portofino-m.jpg
   - lambo-huracan-evo.jpg
   - lambo-huracan-sto.jpg
   - lambo-urus.jpg
   - porsche-911-gt3.jpg
   - porsche-911-targa.jpg
   - audi-r8.jpg
   - audi-rs6.jpg
   - range-rover-sport.jpg
   - range-rover-defender.jpg
   - mercedes-g63.jpg
   - maybach-gls600.jpg
   - mercedes-e-cabrio.jpg
   - mini-cooper-s-cabrio.jpg
   - mercedes-v-class.jpg

If an image is missing, the page will show a neutral placeholder automatically.
You can adjust prices and filenames in assets/fleet.json.
